Environment constraints:

CRITICAL ISSUES:
1. Conda TOS blocks default channels: CondaToSNonInteractiveError
2. CMake not found on PATH (required for some R package builds)
3. xvfb (X virtual framebuffer) missing - required for graphics in headless environment
4. R packages missing: FactoMineR, limma (Bioconductor)
5. PackagesNotFoundError for xorg-xvfb, r-limma, bioconductor-limma

INSTRUCTIONS:
- Do NOT use `apt-get` (blocked).
- Use conda (env `base`) with conda-forge AND bioconda channels.
- Some packages may not exist in conda - use BiocManager for Bioconductor packages.

XVFB ALTERNATIVE:
The package 'xorg-xvfb' may not be available. Alternatives:
1. Try running without display: `DISPLAY=:99 Rscript Main.R`
2. Use headless graphics backend if supported
3. If graphics are required and xvfb unavailable, document the limitation

CMAKE INSTALLATION:
```
conda install -c conda-forge cmake
which cmake  # verify
```

LIMMA PACKAGE (Bioconductor):
If r-limma is not available via conda, install via BiocManager:
```R
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager", repos="https://cloud.r-project.org")
BiocManager::install("limma", ask=FALSE, update=FALSE)
```

FACTOMINER:
```
conda install -c conda-forge r-factominer
```

SETUP:
1. Create writable R library: `mkdir -p ./.R/library && export R_LIBS_USER=./.R/library`
2. Create results directory: `mkdir -p ./results`
3. Install dependencies as above

EXECUTION:
If xvfb is available:
```
xvfb-run Rscript Main.R
```

If xvfb is NOT available, try:
```
Rscript Main.R
```
(Graphics output may fail, but computations should proceed)

OUTPUT: Write all results to `./results` directory.
