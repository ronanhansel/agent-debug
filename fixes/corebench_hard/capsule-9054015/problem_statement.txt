Install R locally and run the capsule’s R scripts entirely inside the repo.

1. Run `mamba install -y -c conda-forge r-base=4.3 r-essentials` (or `conda install`) in the active env. Install the packages referenced in the task via CRAN, e.g.  
   `Rscript -e "install.packages(c('tidyverse','ggplot2','dplyr','readr','rmarkdown','knitr'), repos='https://cloud.r-project.org', dependencies=TRUE)"`, adding any additional `library()` calls you see.
2. Set a writable package path: `mkdir -p r_libs && export R_LIBS_USER="$(pwd)/r_libs"`.
3. Create a workspace results directory (e.g., `mkdir -p results`) and adjust any script arguments or env vars so outputs go to `results/` rather than `../results`.
4. Run the capsule’s `Rscript` commands from the repo root and use the generated artifacts to answer the rubric questions.
